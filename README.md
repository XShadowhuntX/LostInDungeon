# 🏰 LostInDungeon

A text-based dungeon adventure game written in C#.

You wake up in a dark dungeon with no memory of how you got there.
Make choices, survive encounters, and explore the dungeon step by step.

---

## 🎮 Features

* Character creation (Name & Gender)
* Different traits depending on your choice:

  * **Male** → strong, tall, intimidating
  * **Female** → quick, small, beauty
* Multiple decision paths
* Combat situations and survival mechanics
* Inventory elements (keys, bow, arrows, strength upgrades)
* Expandable dungeon system

---

## 🧠 Gameplay

The game is based on choices you type into the console.

Examples:

* `LEFT` / `RIGHT`
* `SNEAK` / `SMASH`
* `RUN` / `SMASH`
* `ATTACK` / `GRAB` / `TALK`

Your decisions influence:

* Survival
* Health
* Items
* Story progression

---

## ⚙️ Requirements

* .NET SDK (recommended: .NET 6 or higher)
* Any C# compatible IDE (e.g. Visual Studio Code, Visual Studio)

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/your-username/LostInDungeon.git
```

2. Navigate into the project folder:

```
cd LostInDungeon
```

3. Run the game:

```
dotnet run
```
